# Sistema de Gerenciamento de Funcionários (Java)

Aplicação de console desenvolvida em Java para gerenciar funcionários, permitindo cadastro, validação de ID e aumento de salário.

##  Funcionalidades
- Cadastro de funcionários
- Validação de ID (não permite IDs duplicados)
- Aumento de salário por porcentagem
- Listagem de todos os funcionários

##  Conceitos utilizados
- Programação Orientada a Objetos (POO)
- Encapsulamento
- Listas (`ArrayList`)
- Estruturas de repetição (`for`, `while`)
- Estruturas condicionais (`if/else`)

## 🛠 Tecnologias
- Java

## Como executar
1. Compile o projeto
2. Execute a classe `Program`
3. Siga as instruções no terminal
